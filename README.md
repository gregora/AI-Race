# Simple Box2D and SDL2 wrapper

## Dependencies
* Box2D (Headers already in `include/` and static library already in `lib/`)
* SDL2

## Compile on Linux
* Run `sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev`
* Run `compile.sh`
* Run `saves/files.sh`
* Train your network before using it
